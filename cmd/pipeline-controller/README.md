# pipeline-controller

Kubernetes controller for managing CI/CD pipelines.

## Overview

Pipeline-controller manages pipeline resources and orchestrates pipeline execution based on defined workflows and dependencies.

## Usage

```bash
pipeline-controller \
  --kubeconfig=/path/to/kubeconfig
```

## Options

- `--kubeconfig`: Path to kubeconfig file
- Additional controller-specific options
